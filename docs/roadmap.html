목표: "코드 실행"이 아니라 "수식 의미 + 구현 연결"까지 설명 가능해지는 것

운영 규칙 (docs 중심)

  1. 개념 글은 docs/posts/<분류>/<주제>_concept.html로 관리
  2. 실험 글은 docs/posts/<분류>/<주제>_experiment.html로 관리
  3. 새 글 추가 후 docs/index.html, docs/posts_index.json, docs/roadmap.html 링크 정합성 점검
  4. 각 단계 완료 조건: "핵심 수식을 말로 설명 + 간단 코드 재현" 둘 다 충족

0단계. 수학 최소 기초 (진입 필수)

목표: 벡터/행렬 연산, 확률, 미분을 신경망 문맥에서 읽을 수 있어야 함

0-1 선형대수

  • 핵심: 벡터, 행렬곱, 전치, 노름, 내적
  • 핵심식: y = Wx + b, ||x||_2
  • 완료조건: 차원 오류 없이 Wx+b 계산/설명

관련 문서: 0-1 선형대수 기초 (기초)

0-2 확률/통계

  • 핵심: 조건부확률, 기댓값, 분산, 로그우도
  • 핵심식: P(y|x), \log p(x)
  • 완료조건: Cross-Entropy가 왜 필요한지 설명

관련 문서: 0-2 확률/통계 기초 (기초)

0-3 미분/연쇄법칙

  • 핵심: 편미분, 기울기, 연쇄법칙
  • 핵심식: \frac{\partial L}{\partial w}, \frac{d}{dx}f(g(x))
  • 완료조건: 역전파의 "뒤에서 앞으로" 이유 설명

관련 문서: 0-3 미분/역전파 기초 (기초)

0-4 확률/통계 최소셋

  • 핵심: 베이즈 관점, 기댓값/분산, 로그우도
  • 핵심식: p(\theta|D) \propto p(D|\theta)p(\theta), \mathbb{E}[X]
  • 완료조건: \"우도 vs 사후확률\" 차이를 말로 설명

관련 문서: 0-4 확률/통계 심화 기초 (기초)

0-5 수치 안정성 기초

  • 핵심: overflow/underflow, 안정한 softmax 계산
  • 핵심식: softmax(x_i)=\frac{e^{x_i-\max(x)}}{\sum_j e^{x_j-\max(x)}}
  • 완료조건: log-sum-exp trick의 필요성 설명

관련 문서: 0-5 수치 안정성 기초 (기초)

  4. 역전파: Local gradient가 곱으로 누적되는 구조 | 관련 문서 (기초)
