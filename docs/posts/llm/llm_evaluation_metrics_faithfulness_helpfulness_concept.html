주제: LLM Evaluation Metrics (Faithfulness / Helpfulness 등)

분류: llm

타입: concept

난이도: 중급

LLM 평가는 단일 정확도가 아니라 다면적 지표가 필요합니다.

  • Faithfulness(근거 충실성)
  • Helpfulness(유용성)
  • Harmlessness(안전성)

RAG에서 중요 지표

  • Context precision/recall
  • Answer correctness
  • Groundedness

실무 팁

  • 자동 지표 + 사람 평가 병행
  • 평가셋 고정으로 회귀 테스트
