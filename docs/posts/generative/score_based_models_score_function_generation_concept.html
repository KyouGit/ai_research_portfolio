주제: 8-4 Score-Based Models (Score Function 기반 생성 모델)

분류: generative

타입: concept

난이도: 중상

선수지식: 있음 — 확률분포, DDPM, Gradient

문제 설정

생성 모델의 목표는 실제 데이터 분포를 학습하는 것입니다.

x ~ p_data(x)

하지만 실제 데이터 분포 p(x)는 보통 직접 계산하기 어렵습니다.

Score-based 모델은 다음 아이디어를 사용합니다.

"분포 자체가 아니라 분포의 gradient를 학습하자"

1. Score Function

Score function은 다음과 같이 정의됩니다.

s(x) = ∇_x log p(x)

기호 의미

  • p(x) : 데이터 확률분포
  • log p(x) : 로그 확률
  • ∇_x : x에 대한 gradient
  • s(x) : score function

왜 필요한가

확률 밀도의 방향 정보를 제공합니다.

직관

  • score는 "데이터 밀도가 증가하는 방향"

주의점

p(x)를 직접 계산하지 않고 gradient만 학습합니다.

2. Score Matching

모델은 score function을 근사합니다.

s_θ(x) ≈ ∇_x log p(x)

loss:

L = E[ || s_θ(x) - ∇_x log p(x) ||² ]

기호 의미

  • s_θ(x) : 모델이 예측한 score
  • θ : 모델 파라미터

왜 필요한가

데이터 분포의 gradient를 학습합니다.

3. Noise Perturbation

실제 데이터는 보통 노이즈를 추가하여 학습합니다.

x̃ = x + σ ε

기호 의미

  • x : 원본 데이터
  • ε : Gaussian noise
  • σ : noise scale

모델은 다음을 학습합니다.

s_θ(x̃, σ)

왜 필요한가

다양한 noise 수준에서 score를 학습합니다.

4. Annealed Score Matching

여러 noise level을 사용합니다.

σ₁ > σ₂ > ... > σ_K

loss:

L = Σ λ_k E[ || s_θ(x̃_k) - score_k ||² ]

왜 중요한가

모델이 다양한 noise 수준에서 안정적으로 학습합니다.

5. Langevin Dynamics

score function을 사용하여 샘플을 생성합니다.

x_{t+1} = x_t + η s_θ(x_t) + √(2η) z

기호 의미

  • η : step size
  • z : Gaussian noise
  • s_θ(x_t) : score estimate

왜 필요한가

score 방향으로 샘플을 이동시켜 데이터 분포로 수렴합니다.

직관

  • noise → 점점 데이터 분포로 이동

6. Diffusion과 Score-Based 관계

Score-based 모델과 diffusion 모델은 매우 밀접합니다.

모델 핵심 학습 대상
DDPM noise ε
Score-based score ∇ log p(x)

두 모델은 사실상 같은 framework로 해석됩니다.

7. Sampling 과정


Noise sample
   ↓
Score estimate
   ↓
Langevin update
   ↓
Iterate
   ↓
Generated sample

8. Score-Based 모델 장점

  • stable training
  • high-quality generation
  • theoretical foundation strong

9. 한계

  • sampling 속도 느림
  • 많은 diffusion step 필요
  • compute cost 높음

코드-수식 연결

개념 코드 설명
noise 추가 x_t = x + sigma * noise perturbed sample
score prediction score = model(x_t) score estimation
Langevin update x = x + step * score + noise sampling step

자주 하는 오해 5개

  • score function은 확률 자체라고 생각한다
  • score-based 모델은 GAN 변형이라고 생각한다
  • noise perturbation은 학습에 필요 없다고 생각한다
  • Langevin dynamics는 optimizer라고 생각한다
  • score-based 모델과 diffusion은 완전히 다른 모델이라고 생각한다
