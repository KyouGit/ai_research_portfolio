신약개발 특화 생성 모델 이해

난이도: 중상

태그: drug_discovery,generation,vae,diffusion,flow

신약개발 생성 모델의 목표는 그럴듯한 분자를 아무거나 만드는 것이 아니라, 유효하고 합성 가능하며 표적과 잘 맞을 가능성이 있는 분자를 제약 조건 아래 생성하는 것이다.

1. VAE 기반 분자 생성

JT-VAE는 분자를 트리나 서브구조 단위로 나눠 latent space에서 조작 가능하게 만든다.

2. Diffusion 기반 분자 생성

DDPM/DDIM 아이디어를 분자와 3D 구조에 확장한 것이 DiffSBDD, TargetDiff, GeoDiff 같은 계열이다.

3. Flow 기반 생성

정규분포와 데이터 분포 사이의 가역 변환을 학습하는 접근이다.
