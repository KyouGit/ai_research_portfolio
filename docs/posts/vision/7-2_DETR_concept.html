[DETR] 개념 설명

난이도: 초중급

태그: Vision,컴퓨터비전

DETR은 객체탐지를 Transformer와 set prediction 관점으로 재정의한 모델입니다.

1단. 문제 정의

해당 분야의 핵심 문제를 해결하기 위한 모델입니다.

2단. 기존 한계

이전 접근의 속도/품질/확장성 한계를 개선합니다.

3단. 핵심 아이디어

모델 구조와 학습 목적을 바꿔 성능을 개선합니다.

4단. 비용/리스크

연산 비용과 데이터 품질 의존성 관리가 필요합니다.

5단. 실무 적용

태스크와 latency 예산에 맞춰 도입 판단을 해야 합니다.
